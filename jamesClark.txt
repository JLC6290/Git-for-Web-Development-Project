    1. What is Semantic HTML? 
    The elements of HTML which give meaning to the content. The content is partitioned into meaningful categories, which are standardized for interpretation by machines.
    2. What is HTML used for? 
    HTML is used to structure the content on web pages in a standardized way for machines to interpret.
    3. What is an attribute and where do we put it? 
    An attribute goes in the opening tag of an HTML element. It allows for heirarchical specificity which determines how different elements will behave.
    4. What is the h1 tag used for? How many times should I use it on a page?
    The H1 tag will typically be used only once, for the most important title on a web page.
    5. Name two tags that have required attributes
    The link tag <a href="url">, and the img tag <img src="url">. These tags exist to link content to the web page.
